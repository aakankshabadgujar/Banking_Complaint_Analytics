## Useful Pandas Functions

head()

Displays the first few rows.

---

info()

Displays:

- Number of rows
- Data types
- Missing values

---

describe()

Displays statistical summaries for numeric columns.

---

columns

Lists all column names.

---

## Why Inspect Data?

Before cleaning data, an analyst should understand:

- Dataset size
- Data types
- Missing values
- Column names
- Overall data quality

---

## Interview Tip

Data inspection is the first step in Exploratory Data Analysis (EDA) and helps identify issues before transformation.
